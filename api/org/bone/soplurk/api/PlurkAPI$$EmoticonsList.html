EmoticonsList

case class EmoticonsList(custom: List[CustomIcon], recruited: Map[Int, List[Emoticon]], karma: Map[Int, List[Emoticon]]) extends Product with Serializable

Reprsented emoticons that current user can use.

The recruited, karma is a Map[Int, List[Emoticon]], where key is the lower bound of recrutied count or karma.

For example, if your karma looks like the following:

Map(
 0 -> List(Emoticon(""":-))""",  "http://statics.plurk.com/4.gif"),
           Emoticon(""":-)""",   "http://statics.plurk.com/5.gif")),

25 -> List(Emoticon("""(:""",        "http://statics.plurk.com/6.gif"),
           Emoticon("""(K)""",       "http://statics.plurk.com/7.gif"),
           Emoticon("""(angry)""",   "http://statics.plurk.com/8.gif"),
           Emoticon("""(annoyed)""", "http://statics.plurk.com/9.gif"))
)

It means you need karma >= 0 to use :-)) and :-), karma >= 25 to use (:, (K), (angry), (annoyed).

custom

Custom icons uploaded by user.

recruited

key is the lower bound of recrutied count to use icons in the corresponding value.

karma

key in the lower bound of karma to use icons in the corresponding value.
